# Research archive

This directory is durable memory for product and technical discovery.

## Convention

```text
YYYY-MM-DD-short-topic/
├── BRIEF.md
├── EVIDENCE.md
├── OPPORTUNITY.md
├── EXPERIMENT.md
└── RETROSPECTIVE.md
```

Copy files from `../templates/`.

## Rules

- One folder represents one decision, not an unlimited market report.
- Keep source URL, publication date, and access date beside claims.
- Separate facts, user statements, source interpretation, and agent inference.
- Preserve meaningful contradiction and rejected concepts.
- Do not store secrets, unnecessary personal data, or raw customer transcripts
  without explicit permission and a retention policy.
- Link promoted research to its issue, implementation plan, PR, and result.
- Add the final status to `INDEX.md`.
